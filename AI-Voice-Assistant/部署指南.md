# 大创项目网站部署指南

## 一、前提条件

1. **确保已安装Python**
   - 版本要求：Python 3.x
   - 验证方式：打开命令提示符，输入 `python --version` 或 `python3 --version`

2. **确认网站文件位置**
   - 网站文件目录：`c:\Users\29717\Desktop\大创网站`

## 二、部署步骤

### 步骤1：打开命令提示符（CMD）

- 按下 `Win + R` 组合键
- 输入 `cmd`，回车打开命令提示符

### 步骤2：进入网站目录

```bash
cd "c:\Users\29717\Desktop\大创网站"
```

### 步骤3：启动本地HTTP服务器

```bash
python -m http.server 8080
```

**执行结果**：
- 命令行显示服务器已启动，监听在端口8080
- 保持此命令窗口打开，不要关闭

### 步骤4：获取本机IPv4地址

**方法一：使用ipconfig命令**
```bash
ipconfig | findstr "IPv4"
```

**方法二：使用hostname命令**
```bash
hostname -I
```

**示例输出**：
```
IPv4 Address. . . . . . . . . . . : 10.6.134.88
```

### 步骤5：测试本地访问

在电脑浏览器中输入以下地址测试：
```
http://localhost:8080
```

或使用本机IP：
```
http://10.6.134.88:8080
```

## 三、手机访问步骤

### 前提条件
- 手机和电脑必须连接**同一个校园网/Wi-Fi**

### 访问方式

1. **打开手机浏览器**
2. **输入网址**（替换为你获取的实际IP）：
   ```
   http://10.6.134.88:8080
   ```

### 页面访问路径

| 页面 | 网址 |
|------|------|
| 主页 | http://10.6.134.88:8080/index.html |
| 详情介绍 | http://10.6.134.88:8080/detail.html |
| 项目展示 | http://10.6.134.88:8080/show.html |
| 联系我们 | http://10.6.134.88:8080/tell.html |

## 四、停止服务器

1. 回到运行服务器的命令提示符窗口
2. 按下 `Ctrl + C` 组合键
3. 服务器停止运行

## 五、常见问题及解决

### 问题1：端口8080被占用

**错误信息**：
```
OSError: [WinError 10048] 通常每个套接字地址(协议/网络地址/端口)只允许使用一次
```

**解决方法**：
- 使用其他端口，如8000：
  ```bash
  python -m http.server 8000
  ```
- 然后访问网址改为：`http://10.6.134.88:8000`

### 问题2：手机无法访问

**可能原因**：
1. 手机和电脑不在同一网络
2. 电脑防火墙阻止了访问
3. 服务器未正常启动

**排查步骤**：
1. 确认手机和电脑连接同一个Wi-Fi
2. 在电脑上测试 `http://localhost:8080` 是否能访问
3. 尝试关闭电脑防火墙后重试

### 问题3：Python命令找不到

**错误信息**：
```
'python' 不是内部或外部命令，也不是可运行的程序
```

**解决方法**：
- 使用完整路径，例如：
  ```bash
  "C:\Program Files\Python39\python.exe" -m http.server 8080
  ```
- 或重新安装Python并勾选"Add Python to PATH"选项
